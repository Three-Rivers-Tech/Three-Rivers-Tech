# Community-Focused Imagery for Three Rivers Tech

This directory contains images that support the community-focused transformation of the Three Rivers Tech website.

## Image Strategy

### Local Connection Images
- **turtle-creek-river-tech.jpg**: Hero background showing Turtle Creek river with subtle technology overlay
- **turtle-creek-borough.jpg**: Local landmarks that Turtle Creek residents would recognize
- **grant-street-storefront.jpg**: The actual storefront at 124 Grant Street (when available)

### Community Service Images

### Image Requirements
- High resolution (1920px+ for hero images)
- Diverse, relatable people (families, seniors, small business owners)
- Technology shown in everyday, accessible contexts
- Warm, welcoming atmosphere
- Professional but approachable feel

### Accessibility Considerations
- All images must have descriptive alt text
- Sufficient contrast for text overlays
- Responsive sizing for all devices
- Optimized loading for various connection speeds

## Implementation Notes

The CommunityImagery component handles:
- Fallback to appropriate stock images
- Responsive sizing and optimization
- Accessibility features
- Local community indicators
- Proper overlays for text readability
