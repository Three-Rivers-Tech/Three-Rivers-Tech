# Three Rivers Tech website
- Static Next.js 15.5.2 + React 19 site that serves Turtle Creek, PA with accessible, community-focused tech services messaging.
- Resides in website/ with App Router, strict TypeScript, Tailwind utility classes, and shared UI components under src/components/ui/.
- Deployed as a static export to Cloudflare Pages; avoid server-only/runtime-dependent patterns.
